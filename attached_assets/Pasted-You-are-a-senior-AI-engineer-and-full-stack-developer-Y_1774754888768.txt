You are a senior AI engineer and full-stack developer. Your task is to design and build a complete working prototype for the following project:

PROJECT TITLE:
Cross-Border Sentinel: Multi-Modal Fake News Detection System

OBJECTIVE:
Build a simplified but functional multi-modal fake news detection system that analyzes both text and images and outputs a credibility score.

IMPORTANT CONSTRAINTS:

* This is a SOLO student project (4th semester CS level)
* The system MUST be simple, modular, and buildable within 2–3 weeks
* Focus on WORKING PROTOTYPE, not research-level complexity
* Avoid overly complex deep learning pipelines
* Use pre-trained models wherever possible

---

SYSTEM OVERVIEW:

The system will take:

1. News text input
2. Optional image input

And output:

* Credibility Score (0–100)
* Prediction (Real / Fake)
* Explanation (why it was classified this way)

---

ARCHITECTURE (3 MAIN MODULES):

1. TEXT ANALYSIS MODULE (CORE MODULE)

Goal:
Classify news as real or fake using NLP

Implementation:

* Use Hugging Face Transformers
* Model: bert-base-uncased or distilbert-base-uncased
* Fine-tune on fake news dataset

Dataset options:

* FakeNewsNet (recommended) → contains text + images ([Nature][1])
* LIAR dataset (backup for text-only) ([Hugging Face][2])

Steps:

* Preprocess text (lowercase, remove noise)
* Tokenize using tokenizer
* Train binary classifier
* Output probability score

---

2. IMAGE VERIFICATION MODULE (LIGHTWEIGHT)

Goal:
Perform basic image validation (NOT advanced forensics)

Implementation:

* Extract metadata (EXIF)
* Check:

  * Creation date
  * Missing metadata
* Flag suspicious images

Optional (if time permits):

* Use simple CNN or pre-trained model for image classification
* OR detect anomalies (blur, artifacts)

NOTE:
Do NOT implement GAN detection or deepfake detection (too complex)

---

3. CREDIBILITY SCORING MODULE

Goal:
Combine outputs into final score

Logic:

* NLP confidence score (70% weight)
* Image flags (30% weight)

Example:
final_score = (0.7 * text_score) + (0.3 * image_score)

Output:

* Score (0–100)
* Label (Real / Fake)
* Reason explanation
